Jekyll is a simple and powerful static site generator that allows you to build websites using plain text files.
It is widely used with GitHub Pages for easy deployment.

In this project, I used Jekyll to create a multi-page website that includes a homepage, about page, projects section, and blog.

### Key Features of Jekyll:
- Uses Markdown for content creation
- Supports layouts and templates
- Automatically generates static pages

This makes it a great tool for beginners learning web development.
