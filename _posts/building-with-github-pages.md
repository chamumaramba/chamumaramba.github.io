GitHub Pages allows developers to host static websites directly from a GitHub repository.
It integrates seamlessly with Jekyll, making deployment simple and efficient.

### What I Learned:
- How to push code to a GitHub repository
- How GitHub Pages automatically builds and hosts the site
- How to update content and see changes live

This workflow is useful for creating portfolios, documentation sites, and blogs.
