Projects

This page highlights projects developed using GitHub Pages and the Jekyll static site generator. The focus is on building and deploying static websites using modern tools and workflows.

Project 1: Personal Website with Jekyll

This project involves creating a multi-page static website using Jekyll and hosting it on GitHub Pages. It demonstrates how to structure a site using layouts, Markdown, and reusable components.

  • Built with Jekyll layouts and Markdown
  • Deployed using GitHub Pages
  • Includes Home, About, Projects, Contact, and Blog pages

Project 2: Jekyll Blog Implementation

A blogging feature implemented within the Jekyll site to showcase how posts can be created, organized, and automatically generated.

  • Uses Jekyll collections and posts
  • Automatic page generation for blog entries
  • Structured content using Markdown

Project 3: Static Site Deployment Workflow

This project focuses on the process of deploying a static website using GitHub Pages, including version control and continuous updates.

  • Version control using Git and GitHub
  • Automatic deployment via GitHub Pages
  • Efficient content updates and maintenance

These projects demonstrate practical experience in building and deploying static websites using GitHub Pages and Jekyll.
