About This Website

Welcome to my GitHub Pages website! This site was created as part of an academic assignment for the Educational Legal Frameworks, Leadership and Management course. It demonstrates the use of Jekyll for static website generation and GitHub Pages for hosting.

Purpose of the Project

The main purpose of this website is to provide hands-on experience in building and deploying multi-page static websites. It also showcases how educational content and assignments can be structured professionally online.

Features

  • Multi-page structure with clear navigation
  • Integration of a Jekyll theme for consistent styling
  • Use of Markdown and HTML for content creation
  • Practical examples from school visits and educational research

Skills Gained

  • Setting up a Jekyll site and choosing an appropriate theme
  • Version control using Git and GitHub
  • Writing semantic HTML for structured content
  • Deploying a static site on GitHub Pages

Future Improvements

In the future, this website can be extended with:

  • Interactive elements using JavaScript
  • Additional pages for case studies or school visit reports
  • Visual media, including photos and diagrams from educational activities
  • Better accessibility and responsive design enhancements
